# handwritten-number-recognition
A Machine Learning project that recognizes handwritten numbers from 1 to 9 using image classification techniques.

This project was created as a collaborative study project to explore:

-supervised learning,
-image preprocessing,
-neural networks,
-Computer vision fundamentals.


# Project Objective

The goal of this project is to build a model capable of identifying handwritten digits from images.

The model receives an image containing a handwritten number and predicts which digit it represents.


## Dataset

- MNIST

## Features

- Image preprocessing
- Handwritten digit classification
- Model training and evaluation
- Prediction visualization
